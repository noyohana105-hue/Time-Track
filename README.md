# Shift Tracker App - Upgraded Version

## Overview
Shift Tracker App is an Android application designed to help users track their shifts, work hours, and earnings efficiently. Whether you're working part-time, freelancing, or managing multiple jobs, this app provides a simple solution to keep track of your work schedule, hours, and total salary.

## New Features & Enhancements
- **Salary Calculation**: Automatically calculates your total earnings based on your logged shifts and defined hourly rate.
- **Job Settings Management**: Easily update your job name, hourly rate, and start date through a dedicated settings screen accessible from the main dashboard.
- **Improved UI/UX**: Features a modern, clean login screen with a professional gradient background and card-based layout.
- **Easy Navigation**: Added "Back" buttons to all configuration screens for a smoother user experience.
- **Secure Logout**: A dedicated logout feature that clears local session data and returns to the login screen.
- **Shift Tracking**: Record your shifts by entering start and end times with automatic duration calculation.
- **Firebase Integration**: 
    - **Authentication**: Secure login and registration.
    - **Cloud Firestore**: Real-time data storage and synchronization across devices.

## Installation
1. Clone the repository to your local machine.
2. Open the project in Android Studio.
3. Ensure you have a valid `google-services.json` in the `app/` directory.
4. Build and run the app on an Android device or emulator.

## Usage
1. **Sign Up/Login**: Create a new account or log in with existing credentials.
2. **Setup Job**: On first login, you'll be prompted to set your job name and hourly rate.
3. **Log Shifts**: Use the "+" button to add your entrance and leaving times.
4. **Track Earnings**: View your total hours and calculated salary directly on the main dashboard.
5. **Manage Settings**: Use the ⚙️ icon to update your job details or hourly rate anytime.
